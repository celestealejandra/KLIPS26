# Korean Labor and Income Panel Survey (KLIPS)
This repository contains R codes for extracting and analizing data provided by the Korean Labor and Income Panel Survey (KLIPS). For the purpose of
current research, in this repository we work with the 26th wave conducted in 2023 in South Korea by the Korean Labor Institute (KLI). 

Este repositorio contiene los códigos en R para la extracción y el análisis de los datos proporcionados por la Korean Labor and Income Panel Survey (KLIPS). Para la presente investigación, se utiliza la 26.ª ola de la encuesta, levantada en 2023 en Corea del Sur por el Centro de Investigaciones Laborales de Corea del Sur (KLI). 

 **Download URL:** https://www.kli.re.kr/board.es?mid=a50601000000&bid=0058&act=view&list_no=145600&tag=&nPage=1
