# Korean Labor and Income Panel Survey (KLIPS)
This repository contains R codes for extracting and analizing data provided by the Korean Labor and Income Panel Survey (KLIPS). For the purpose of
current research, in this repository we work with the 26th wave conducted in 2023 in South Korea. 

 **Download URL:** https://www.kli.re.kr/board.es?mid=a50601000000&bid=0058&act=view&list_no=145600&tag=&nPage=1
